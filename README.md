# VisionCare EMR - Luna Eye Hospital

A standalone LAN-based Electronic Medical Records (EMR) and clinic operations system built for Luna Eye Hospital. 

## Key Features
- **Ophthalmology Workflow**: Specialized clinical documentation for eye care.
- **Billing & Transactions**: Modernized POS-style billing with support for split payments and insurance.
- **Inventory Management**: Real-time stock tracking for drugs, lenses, and consumables.
- **Reprint Management**: Secure audit-logged receipt and report reprints.
- **Multi-Role Access**: Dedicated dashboards for Admin, Reception, Nurse, and Consultant.

## Technology Stack
- **Frontend**: React 19, TypeScript, Vite, TailwindCSS, Lucide React.
- **Backend**: Node.js, Express 5.
- **Database**: SQLite3.
